# PTIA-lab01

This repository contains the development of Laboratory 1 (of 4) for the course Principles and Technologies of Artificial Intelligence.

## Getting Started

These instructions will help you set up the project on your local machine for development and testing purposes.

### Prerequisites

You need the following software installed:

- Python 3.x
- Libraries:
  - `pandas`
  - `scikit-learn`

Install the required libraries using pip:

```bash
pip install pandas scikit-learn
```

Clone the repository

```bash
git clone https://github.com/your-username/PTIA-lab01.git
cd PTIA-lab01
```
